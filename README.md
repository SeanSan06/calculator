# Calculator
A simple add, subtract, multiply, and divide calculator that display results on the screen dynamically.
Try it: [https://seansan06.github.io/calculator/](https://seansan06.github.io/calculator/)

## How its made
Technology used: HTML, CSS, JavaScript, and Github pages.

HTML: Used to add the basic content such as the caculators buttons and screen to display the operations.

CSS: CSS and grid to allign all the buttons to where they need to be.

JavaScript: Used JavaScript to add the logic for all the basic operations and dynamically display them on the screen.

Github pages: Used to host the landing page.
